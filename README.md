  Our project is a distributed smart smoke detection and emergency response system built using four Arduino microcontrollers. 
  Instead of relying on a single board, the system distributes tasks across multiple Arduinos to simulate a realistic building safety network. 
  The system detects dangerous temperature levels that may indicate a fire and responds automatically. 
  When triggered, it activates LED warnings, displays alerts on an LCD, sounds an alarm, and turns on a ventilation fan. 
  Motion detection monitors occupancy, while a keypad and push button allow users to arm, disarm, or reset the system, 
  demonstrating coordinated embedded control in safety applications.
